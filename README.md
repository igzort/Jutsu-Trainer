# Jutsu-Trainer
Тренажер Ручных Печатей
[![Python Version](https://shields.io)](https://python.org)
[![Framework](https://shields.io)](https://kivy.org)

Интерактивный игровой тренажер на Python, вдохновленный вселенной Наруто. Проект помогает начинающим шиноби освоить и быстро воспроизводить комбинации ручных печатей (Hand Seals) для применения известных техник (дзюцу), таких как Чидори, Расенган, Катон, теневое клонирование и др.

## ✨ Особенности проекта
* **Продвинутый GUI на Kivy:** Кастомные кнопки печатей на базе `ButtonBehavior` и `AnchorLayout` с отображением горячих клавиш прямо поверх текстуры печати.
* **Комбо-система:** Отслеживание и валидация правильной последовательности нажатий в реальном времени.
* **Экзамен на ранг (Сложность):** Три игровых режима, жестко ограничивающих время на каст:
  * 🟢 **Чунин** — Лимит времени: 10.0 сек.
  * 🔵 **Дзёнин** — Лимит времени: 4.0 сек.
  * 🔴 **Каге** — Лимит времени: 1.8 сек.
* **Циклический пулл техник:** Игровой движок использует алгоритм «мешка без повторений» (псевдорандом), что исключает появление одного и того же дзюцу два раза подряд.
* **Сочные эффекты при ошибках:** При превышении лимита времени (`TIMEOUT`) или неверной клавише (`MISCLICK`) экран начинает трястись (Screen Shake), сопровождаясь двойной пульсирующей красной рамкой.
* **Аудио-визуальный каст:** Пошаговое анимирование складываемых печатей, кастомный декодер для воспроизведения финальных GIF-эффектов дзюцу и менеджер звука (`AudioManager`) для микширования эффектов и фоновой музыки.

## 🎹 Раскладка управления (12 Печатей)
Игровое поле и клавиатура привязаны к 12 каноничным печатям:
* `Q` — Крыса | `W` — Бык | `E` — Тигр | `R` — Заяц
* `A` — Дракон | `S` — Змея | `D` — Лошадь | `F` — Овца
* `Z` — Обезьяна | `X` — Птица | `C` — Собака | `V` — Кабан

## 📜 Доступные Дзюцу в Базе
1. **Чидори** (`x ➔ c ➔ f ➔ x ➔ c ➔ f`)
2. **Огненный шар (Катон)** (`a ➔ e ➔ d ➔ e ➔ q ➔ q`)
3. **Расенган** (`v ➔ x ➔ x ➔ r ➔ w ➔ f`)
4. **Теневое клонирование** (`d ➔ f ➔ q ➔ w ➔ e ➔ w`)
5. **Водяное клонирование** (`v ➔ c ➔ a ➔ q ➔ w ➔ s`)
6. **Стихия Дерева (Мокутон)** (`s ➔ a ➔ z ➔ q ➔ x ➔ c ➔ v`)

## 📦 Инструкция по установке и запуску

### Требования
Для работы приложения необходим установленный Python версии **3.10 или выше**.

### Пошаговый запуск

1. **Клонируйте репозиторий с проектом:**
```bash
git clone https://github.com
cd название-репозитория
```

2. **Создайте и активируйте виртуальное окружение:**
```bash
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для macOS/Linux:
source venv/bin/activate
```

3. **Установите необходимые библиотеки:**
Для работы интерфейса и парсинга анимаций требуются `Kivy` и `Pillow`:
```bash
pip install kivy pillow
```

4. **Запустите игру:**
```bash
python jutsu.py
```

## 📂 Архитектура репозитория
* `jutsu.py` — Главная точка входа, инициализирует приложение `JutsuTrainerApp`.
* `interface.py` — Основной графический интерфейс игры (отрисовка элементов, анимация каста, логика тряски экрана).
* `engine.py` — Логическое ядро (проверка цепочек печатей, игровой счет, таймеры сложности).
* `effects.py` — Модуль мультимедиа (`AudioManager` для звуков и `GifDecoder` для покадрового рендеринга GIF через PIL).
* `widgets.py` — Стилизованные кнопки `ImageButton` с текстовым слоем поверх изображений.
* `difficulty_widget.py` — Горизонтальная панель выбора ранга сложности.
* `recipes.py` — База данных техник `NINJUTSU_DATABASE` и карта кнопок.
